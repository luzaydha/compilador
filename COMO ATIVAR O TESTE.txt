=========== COMO ATIVAR O TESTE EM PYTHON ============

1.baixar extensão de python com path.


2.No terminal do VScode, para criar o ambiente virtual.
 (colocar os comandos abaixo)

  1." python -m venv .venv "

  2.ATIVADOR DE AMBIENTE VIRTUAL : .\.venv\Scripts\Activate.ps1

  3.para poder ativar o teste : " pip install pytest "

3.PARA  MOSTRAR O COMPILADOR FUNCIONANDO  

  1. pytest -v

  2. python index.py

DESATIVADOR DE AMBIEMTE VIRTUAL: deactivate

=========== CASO DE ERRO DE SEGURANÇA ============

1.Clique no Menu Iniciar do Windows.

2.Digite PowerShell.

3.Clique com o botão direito em "Windows PowerShell" e selecione "Executar como Administrador".

4.Na janela de Administrador, copie e cole este comando e aperte Enter:

"  Set-ExecutionPolicy RemoteSigned -Scope CurrentUser   "
